# Library Management System (C)

## Overview

This project is a simple Library Management System developed in C. It demonstrates the use of structures, arrays, loops, conditional statements, and string handling to manage book records efficiently.

The application allows users to:

* Add new books with unique IDs, titles, and author names.
* Display all available book records.
* Search for a book using its ID.
* Exit the program through a menu-driven interface.

The project stores book information using a structure (`struct Book`) and maintains records in an array, making it an excellent beginner-level project for understanding data management and menu-driven programming in C.

## Features

* Add Book Records
* Display All Books
* Search Books by ID
* User-Friendly Menu Interface
* Efficient Data Storage Using Structures

## Technologies Used

* C Programming Language
* GCC Compiler
* Visual Studio Code

## Learning Outcomes

This project helped in understanding:

* Structures in C
* Arrays and Data Storage

* Looping and Conditional Statements

* Basic Search Operations
